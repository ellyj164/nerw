## User Journey Examples

### Scenario 1: Non-Logged-In User
1. Visits homepage
2. Sees 4 community cards in footer
3. Clicks "Join Community" on Beginner card
4. Sees message: "Please log in first to join this community" + Login link
5. Redirected to login page
6. After login, returns to homepage
7. Clicks "Join Community" again
8. Button changes to "Joined âœ“"
9. Member count increases by 1

### Scenario 2: Member Posting Discussion
1. User is logged in and joined "Intermediate" community
2. Visits /community/
3. Clicks "Intermediate" tab
4. Sees "Start a New Discussion" form
5. Enters title: "Grammar question about subjunctive"
6. Enters content: "When should I use the subjunctive mood?"
7. Clicks "Post Discussion"
8. Discussion appears at top of list
9. Can click to expand and see comments

### Scenario 3: Non-Member Viewing
1. User visits /community/ (not logged in)
2. Can see all discussions
3. Can read comments
4. Cannot see "New Discussion" form
5. Cannot see "Post Comment" button
6. Sees message: "Join this community to participate"
